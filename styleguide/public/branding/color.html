Core Colors

Built of two core color chords: Blue and Gray.

Neutrals, Black, and White balance the vibrant color scheme and help ensure that layouts do not become too colorful or saturated.

Use neutrals for type and other elements such as keylines, tags, charts, and diagrams. They may also be used as background color fields.

Thin and Shade colors extend the range of available colors, but these should only be used as a supplement to the primary colors.

Each color is listed in its RGB and Hexadecimal values to facilitate accurate reproduction across all types of media.

Logotype Colors

Use the reversed versions when the logotype is placed on a dark background. Please note that the reversed versions are rendered using the tint colors to create the appearance of having the same visual weight overall.
